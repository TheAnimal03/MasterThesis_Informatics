## Installing from a local src tree (source: python tutorials installing packages)

Installing from local src in Development Mode, i.e. in such a way that the project appears to be installed, but yet is still editable from the src tree.

### Unix/macOs
python3 -m pip install -e <path> or python3 -m pip install <path>

### Windows
py -m pip install -e <path> or py -m pip install <path>
